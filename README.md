# Задание 1

## Задача 1

### Текст задачи

Создать список из двух чередующихся символов (символы вводятся с клавиатуры)

### Алгоритм решения



Запросить у пользователя список целых чисел.



Для каждого числа из списка рекурсивно вычислить его первую цифру: взять модуль числа, затем последовательно делить его на 10 до тех пор, пока результат не станет меньше 10 (это и есть первая цифра).



Сформировать новый список из полученных первых цифр и вывести его вместе с исходным списком.

# Задание 2

## Задача 1

### Текст задачи

Сколько раз в списке встречается заданное число?



### Алгоритм решения

Использовать ранее введённый список целых чисел.



Запросить у пользователя число, которое нужно подсчитать.



Рекурсивно обойти список, сравнивая каждый элемент с заданным числом; при каждом совпадении увеличивать счётчик.



Вывести итоговое количество вхождений.
